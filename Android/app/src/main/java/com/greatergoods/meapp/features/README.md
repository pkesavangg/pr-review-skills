# Features Module

Contains feature-specific code organized by feature modules.

## Structure

- **common/**: Shared UI components and logic across features
- **auth/**: Authentication flows (Login, Register, etc.)
- **entry/**: Onboarding and splash screens
- **settings/**: User preferences and settings

Each feature follows the structure:
- model/
- enum/
- strings/
- viewmodel/
- view/
  - screens/
  - components/
